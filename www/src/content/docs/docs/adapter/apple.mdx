This is the Apple adapter.

Use this adapter to authenticate with Apple.

#### Using OAuth

```ts
import { AppleAdapter } from "@uswriting/openauth/adapter/apple";

AppleAdapter({
  clientId: "1234567890",
  clientSecret: "0987654321",
});
```

#### Using OIDC

```ts
import { AppleOidcAdapter } from "@uswriting/openauth/adapter/apple";

AppleOidcAdapter({
  clientId: "1234567890",
  clientSecret: "0987654321",
});
```
</Section>
