<Section type="about">
Configure OpenAuth to use [DynamoDB](https://aws.amazon.com/dynamodb/) as a storage adapter.

```ts
import { DynamoStorage } from "@uswriting/openauth/storage/dynamo"

const storage = DynamoStorage({
  table: "my-table",
  pk: "pk",
  sk: "sk"
})

export default issuer({
  storage,
  // ...
})
```
</Section>
