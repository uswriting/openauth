<Section type="about">
Use this provider to authenticate with Twitch.

```ts {5-8}
import { TwitchProvider } from "@uswriting/openauth/provider/twitch"

export default issuer({
  providers: {
    twitch: TwitchProvider({
      clientId: "1234567890",
      clientSecret: "0987654321"
    })
  }
})
```
</Section>
