<Section type="about">
Use this provider to authenticate with JumpCloud.

```ts {5-8}
import { JumpCloudProvider } from "@uswriting/openauth/provider/jumpcloud"

export default issuer({
  providers: {
    jumpcloud: JumpCloudProvider({
      clientId: "1234567890",
      clientSecret: "0987654321"
    })
  }
})
```
</Section>
