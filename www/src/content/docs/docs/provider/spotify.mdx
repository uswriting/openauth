<Section type="about">
Use this provider to authenticate with Spotify.

```ts {5-8}
import { SpotifyProvider } from "@uswriting/openauth/provider/spotify"

export default issuer({
  providers: {
    spotify: SpotifyProvider({
      clientId: "1234567890",
      clientSecret: "0987654321"
    })
  }
})
```
</Section>
