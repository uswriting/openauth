<Section type="about">
Use this to connect authentication providers that support OIDC.

```ts {5-8}
import { OidcProvider } from "@uswriting/openauth/provider/oidc"

export default issuer({
  providers: {
    oauth2: OidcProvider({
      clientId: "1234567890",
      issuer: "https://auth.myserver.com"
    })
  }
})
```
</Section>
